# project-shop
   
#### 기술 스택 
Visual Studio Code   
React, HTML, CSS, Javascript

#### 요약
남성, 여성, 아동 등 카테고리 별로 상품 조회가 가능한 쇼핑몰.
장바구니에 상품을 추가하고 위시리스트에 상품을 추가할 수 있음.

#### 내용
웹 화면을 실제로 구현하는 퍼블리싱 경험.   
퍼블리싱한 페이지를 리액트를 이용해 컴포넌트화.   
React Hook과 REST API를 활용한 데이터 요청과 응답.
